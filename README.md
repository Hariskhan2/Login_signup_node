# Login Signup Node

This is a simple login/signup application built with Node.js, Express, and MongoDB using Mongoose. It uses Passport.js for authentication and bcryptjs for password hashing.

## Setup

1. Clone the repository: `git clone https://github.com/your-username/Login_signup_node.git`
2. Install dependencies: `npm install`
3. Start the server: `npm start` (or `npm run dev` for development with nodemon)

## Usage

1. Visit `http://localhost:3000` in your browser.
2. Register a new account or log in with an existing one.
3. Enjoy using the application!
